# Assets Folder

Place the following image files in this folder for social sharing:

## Required Images

1. **og-image.png** (1200×630px)
   - Main Open Graph image for the homepage
   - Shows when shared on Facebook, LinkedIn, etc.

2. **og-ai-consulting.png** (1200×630px)
   - Open Graph image for AI Consulting page

3. **og-tech-support.png** (1200×630px)
   - Open Graph image for Tech Support page

4. **og-terry-weber.png** (1200×630px)
   - Open Graph image for Terry Weber page

5. **apple-touch-icon.png** (180×180px)
   - iOS home screen icon

## Recommended Specs

- Format: PNG or JPG
- Dimensions: 1200×630px (OG images), 180×180px (touch icon)
- File size: Under 1MB for fast loading
- Include Josh's name/branding on each image

## Design Tips

- Use the brand colors: Navy (#0A1A2F), Blue (#4DB5FF), White
- Include a professional headshot if available
- Keep text minimal but readable
- Test with Facebook Debugger: https://developers.facebook.com/tools/debug/
